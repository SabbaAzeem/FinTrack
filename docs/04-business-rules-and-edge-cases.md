# Business Rules

## For Transaction Type: Income

An income transaction must have:

- Date
- Transaction Type: Income
- Income Category
- Amount
- Destination Account
- Counter Party
- Payment method

It may optionally have:

- Description or notes

- Expense Category
- Amount

- Payment method

- Description or notes
